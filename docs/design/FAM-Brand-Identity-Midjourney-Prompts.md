## Quick Reference: Style Modifier Cheat Sheet

| Use Case | Aspect Ratio | Key Modifiers |
|---|---|---|
| Icons / Logomarks | `--ar 1:1` | `--style raw`, minimal, vector |
| Horizontal Lockups | `--ar 16:9` or `--ar 3:1` | `--style raw`, professional branding |
| Marketing Heroes | `--ar 16:9` | `--style raw --v 6.1`, cinematic lighting |
| UI Mockups | `--ar 16:10` | `--style raw`, Figma quality, realistic |
| Mobile Screens | `--ar 9:16` | `--style raw`, iPhone mockup |
| Social Square | `--ar 1:1` | `--style raw`, social media graphic |

**Global additions for all prompts:** `dark background #09090B, indigo accent #6366f1, premium SaaS`

## Usage Notes

- **Iterate fast:** Run each prompt, then use `--v` (variations) on the best output before upscaling.
- **Remix for refinement:** Turn on Remix Mode in Midjourney to tweak prompts based on initial results.
- **Combine with inpainting:** Use the best outputs as base images and inpaint specific elements (logo placement, text, etc.).
- **Post-production:** All outputs will need vector tracing (logos), color correction, and typography replacement in Figma/Illustrator. Midjourney generates the visual direction, not the final asset.
- **Version note:** These prompts are written for Midjourney v6.1. Adjust `--v` flag if using a different version.
