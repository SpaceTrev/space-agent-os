# GTM Strategy & Content Playbook

## The Funnel

```
Social Content → Audience → Discord Community → Consulting / SaaS Clients
```

Every piece of content is a funnel entry point. Every Discord member is a warm lead.

## Target Audience

**Primary ICP**: Mexican business owners modernizing operations. Non-technical, results-oriented.
- Pain: manual processes, no AI tooling, behind competitors
- Language: Spanish-first content, accessible framing (no jargon)
- Channels: LinkedIn, Twitter/X, Instagram

## Content Principles

1. **Show, don't tell** — demo agents doing real work beats feature lists
2. **Educational first** — teach AI concepts → build trust → sell platform
3. **Consistency beats virality** — daily presence > occasional spikes
4. **Social proof** — document our own results running the platform

## Content Types

| Type | Platform | Cadence |
|------|----------|---------|
| Agent demos (video) | LinkedIn, Twitter | 3x/week |
| Educational threads | Twitter | Daily |
| Case studies | LinkedIn | 2x/week |
| Community updates | Discord | Daily |
| Long-form posts | LinkedIn | 1x/week |

## Automation Stack

- Content brief → Marketing agent generates drafts
- Drafts → Human review (Pablo) → Scheduled via social MCP tools
- Discord announcements → Auto-posted on milestone completions

## Discord Community

Primary channel: `#claw-chat` — where agents post updates, humans give feedback.
Goal: 1000 engaged members → 50 platform trials → 10 paying clients.
