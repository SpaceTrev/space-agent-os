## TL;DR

TradingView MCP connects Claude AI to TradingView's live chart data via Chrome DevTools Protocol (CDP) — reading actual DOM/code-level values, not screenshots. Claude navigates charts, installs Pine Script strategies, and delivers multi-asset morning briefs via natural language in the terminal. The author extends the original (by "Trades Don't Lie") with a one-shot Claude Code setup prompt, a `morning_brief` command, and a `rules.json` strategy config layer. The key insight: instead of screenshotting a chart, Claude reads the live browser DOM which gives it exact candle prices, indicator values, and real-time updates at any timeframe including 1-second.

## Key Takeaways

- **CDP not screenshots**: TradingView MCP uses Chrome DevTools Protocol to read live TradingView DOM data — not image recognition. Claude gets exact code-level values (OHLC, indicator readings) that update in real-time, even on 1-second charts.
- **One-shot setup via Claude Code**: Copy the author's one-shot setup prompt into Claude Code. It creates `mcp.json`, sets up the MCP server, and handles all configuration automatically. Approve "allow edits for this session."
- **CDP launch command required**: TradingView Desktop must be launched via a specific terminal command that enables CDP. Find `TradingView.app` via Finder, copy it into the command Claude provides, then run `tv_health_check` to confirm it's live.
- **`rules.json` is the strategy brain**: Trading rules (bullish/bearish conditions, entry/exit criteria, timeframes, risk parameters) are stored in `rules.json`. You populate it by talking to Claude — no manual file editing needed.
- **Natural language → Pine Script**: Ask Claude to research a public trader's strategy (e.g. "Van de Pop + Tone Vays MACD with 21/50/200 EMA + RSI") and it writes a complete Pine Script and installs it on the chart. It monitors for compile errors and self-corrects in a loop.
- **`morning_brief` command**: One command sweeps your entire watchlist — Claude checks every ticker against your strategy and delivers a consolidated report. Can be cron-scheduled for continuous alerts.
- **Analyst-strategy injection**: Scrape transcripts from trading analysts (e.g. CoinsBKid, Blockchain Backer), feed them to Claude, and ask it to extract their strategy into a Pine Script. Transforms qualitative analysis into quantifiable chart rules.
- **WhisperFlow for voice input**: The author uses WhisperFlow for voice-to-text — speak naturally to Claude instead of typing.
- **Bug fix included**: The original "Trades Don't Lie" version had a TradingView launch bug; this fork fixes it. The one-shot prompt is the recommended setup path.
- **Multi-asset batch apply**: Add assets to your watchlist once; Pine Script strategies apply to all tickers automatically.
