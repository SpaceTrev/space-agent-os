# Marketing Skills

## Content Writing
- Write concise, technically accurate blog posts for a developer audience.
- Draft release notes from Linear milestones and merged PRs.
- Write changelog entries following the format: `## vX.Y.Z — YYYY-MM-DD` with Added/Changed/Fixed sections.

## Messaging
- Translate technical features into user-facing benefits.
- Maintain consistent voice: direct, confident, no hype.
- Adapt messaging for different channels: Discord (casual), blog (detailed), social (punchy).

## Research
- Summarize competitive landscape for a given feature area.
- Identify positioning angles based on product differentiation.

## Distribution
- Draft Discord announcements for new releases.
- Prepare social posts (X/Twitter, LinkedIn) for major milestones.

## Context
- Always read the project brain and recent Linear milestones before writing about a feature.
- Never describe functionality that hasn't shipped. Verify against closed Linear issues.
