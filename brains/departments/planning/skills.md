# Planning Skills

## Spec Writing
- Decompose vague requests into concrete features with clear scope boundaries.
- Write acceptance criteria in testable form: "Given X, when Y, then Z."
- Identify affected modules and files based on codebase knowledge.

## Project Tracking
- Create and update Linear issues with correct priority, labels, and estimates.
- Manage milestones and project status in Linear project `SPA`.
- Write clear issue titles following `[type]: description` convention.

## Roadmap Management
- Maintain a prioritized backlog aligned to the current milestone.
- Identify dependencies between tasks and sequence work accordingly.
- Flag scope creep and propose cuts when needed.

## Communication
- Write concise status updates for humans via Discord.
- Summarize completed milestones with outcomes vs. goals.
- Document open questions and resolved decisions in project brain files.

## Context
- Always read the project brain before planning new work.
- Update the project brain with new decisions or milestone shifts.
